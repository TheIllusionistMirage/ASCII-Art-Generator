===========================
|   ASCII ART GENERATOR   |
===========================

(C) Copyright Koushtav Chakrabarty a.k.a. The Illusionist Mirage

A simple image to ASCII art generator. Supports all image formats as SFML does.
Enjoy!!!

Compile and run:

g++ main.cpp -std=c++11 -lsfml-graphics -lsfml-window -lsfml-system -o ascii-art-gen
./ascii-art-gen
